# ZonaComix
Repositorio de ZonaComix 

##Doctrine
Genrar entidad:
php app/console doctrine:generate:entity

Generar getters y setters
php app/console doctrine:generate:entities ZonaComix

crear tablas
php app/console doctrine:schema:update --force
